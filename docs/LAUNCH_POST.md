# Launch Kit

## Ready-to-Post X Thread
1. I built **Narrative Treemap**: a no-key, local-first app that turns public media into a real-time narrative intelligence map.
2. It ingests **172 public sources** (global + regional + domain-specific) across tech, science, world, markets, sports, entertainment, health, gaming, security, and programming.
3. Every item is normalized, embedded, clustered into narratives, then scored for momentum, novelty, source diversity, and source quality.
4. Treemap view: bigger tiles = larger narrative footprint, hotter colors = rising momentum.
5. Action Center: converts narratives into watchlist-ready actions with quick inspect flows.
6. Decision Briefing: role-based priorities + explicit execution plans + risk caps (theme limits, daily notional caps, cooldowns, no pyramiding).
7. Works with **zero API keys**: public RSS/JSON/light scraping only.
8. Cached results load instantly; refresh runs in background so the UI stays responsive.
9. Includes seeded demo data, so first run is alive in seconds.
10. Open source, Docker-ready, and built for anyone who wants signal over noise.

## 20s Demo Script
- 0-3s: Open Treemap tab and show the narrative map + trending strip.
- 3-7s: Switch window `24h -> 1h` and toggle `Only rising`.
- 7-11s: Filter one vertical and one source group, then click a hot tile.
- 11-15s: Show the detail drawer (summary, top links, entities, related topics).
- 15-18s: Switch to Decision Briefing and show `Act now` decisions + risk caps.
- 18-20s: Switch to Action Center and end on the full app view.

## Suggested Recording Checklist
- Use a 16:9 window at 1440x900 or 1920x1080.
- Keep cursor movement deliberate and slow.
- Ensure local seed data is visible even before live refresh.
- Capture one source-health error badge to demonstrate resilience.
